# Change Log

All notable changes to the GhostBot VS Code extension.

## [0.2.0]

- Sidebar webview mounting the full GhostBot UI inside VS Code
- `GhostBot: Open` command + `ghostbot-sidebar` activity bar entry
- `GhostBot: New Chat` keybinding (Cmd/Ctrl+Shift+G)
- `GhostBot: Send Selection to Chat` context menu + keybinding
- `GhostBot: Ask About This` CodeLens above functions/classes
- `GhostBot: Open in External Browser` for full-tab browsing
- Configurable `ghostbot.url` (defaults to `https://ghostbot.dev`)
- Optional `ghostbot.openOnStartup` to auto-show the panel

## [0.1.0]

- Initial private build (not published to Marketplace)
- Basic webview wrapping the `ghostbot.dev` URL
